Context

Electron guns are used in high-precision industrial and defense systems at Thales. Predicting how the beam trajectory responds to parameter changes typically requires expensive physical simulations. The goal was to train an ML surrogate model capable of replacing those simulations at inference time.

Since no historical dataset existed, Monte Carlo simulations were run using ANSYS and Persalys to generate synthetic training data across the parameter space. AutoML (H2O) was used to benchmark model families before exploring deep learning architectures with TensorFlow.
